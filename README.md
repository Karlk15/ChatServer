# ChatServer
### Get stated with a clean machine:

> $ cd server

> $ npm install

> $ cd client

> $ npm install

## To run server cmd:

> $ cd server

> $ node chatserver.js

## To run client run cmd:

> $ cd client
 
> $ ng serve

## To test TsLint run cmd:

> $ grunt test 
 
or 
 
> $grunt tslint

## To concatenate and minify run cmd:

> $ cd client
 
> $ ./conCatAndUglify
